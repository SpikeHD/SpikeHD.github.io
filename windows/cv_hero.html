Overview
CVHero as a concept is super simple, it uses OpenCV to detect notes while a song plays, and it automatically emulates keypresses when knows which notes are on screen!
It is written in pure C++, using only the OpenCV library, and Input_Lite
Technical Stuff
The specific implementation uses "template matching", which takes a template image, like an image of a note, and checks to see if there are instances of that template within the image
Obviously a detection cannot be perfect, so it has a "confidence" threshold it uses to determine what is and isn't a note
It uses the Template Matching Correlation method (internally known as TM_CCORR_NORMED)
