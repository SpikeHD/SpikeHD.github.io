NDS_Pong is a very simple project:

It is pong, for the Nintendo DS!


The project is written in C++, using the LibNDS devkitPro library. It only took a few hours to create the base, and it supports local multiplayer!

Above is an early iteration of the game. Score wasn't even calculated, nor displayed

I created this simply to learn something weird. The Nintendo DS was a big part of my childhood, so I though it made a good fit for a quick small project!

The menu at the bottom is controlled by the touchscreen!
