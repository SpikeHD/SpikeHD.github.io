If you couldn't tell, this site is designed to try and replicate a Windows XP desktop environment. I was inspired to do this because my very first computer ran Windows XP! Each desktop icon is a project that I have worked on, and you can click on them to open windows that contain information about them!

Settings

If you didn't like the introduction, or would like to change some other settings, just click the "Settings" desktop icon!
